## Installation

Clone the project

```bash
  git clone https://github.com/fac28/pactportal.git
```

Go to the project directory

```bash
  cd pactportal
```

Install dependencies

```bash
  npm install
```

Seed the database
```bash
  npm run seed
``` 

Start the server

```bash
  npm run dev
```
